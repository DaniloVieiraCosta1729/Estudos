Paragrafos e Quebra de linha.

Não importa como você escreve o paragrafo. Se você está usando as tags <p> e </p>, então o paragrafo será exibido de uma forma padrão, i.e., independente de como você escreveu. (Pelo menos no que diz respeito a quebra de linhas.)

Para quebrar o texto, deve-se usar a tag <br>. Por exemplo: 123
456
789.
