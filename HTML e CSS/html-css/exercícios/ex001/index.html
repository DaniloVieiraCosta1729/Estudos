Agora vou fazer algumas anotações sobre a forma básica de um documento HTLM.

Primeiro nós dizemos o tipo de arquivo (!DOCTYPE html);

Em seguida nós indicamos que vamos começar a escrever o html e, ao mesmo tempo, informamos o idioma (html lang="pt-br");

Agora definimos a cabeça, informando o conjunto de caracteres, o nome e o título. O título será exibido na aba da página onde se encontra o documento;

O passo seguinte é escrever os elementos do corpo, como o título do assunto e os paragrafos.
