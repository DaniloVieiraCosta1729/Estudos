Principais Formatações

Negrito/Destaque

Temos duas formas de usar o Negrito, uma delas é semantica e a outra não.

Não Semantica: A palavra avião está em negrito.
Semantica: a palavra avião está em destaque, i.e., o HTML entende que essa palavra é "especial".

A diferença é que, na primeira, quando usamos <b> estamos lidando apenas com a aparência da letra (pior ainda, estamos usando o html para fazer o serviço do css), enquanto que na segunda, quando usamos <strong>, estamos utilizando um recurso do html que dá um significa de "importante" para a palavra avião.

Itálico/Ênfase

O mesmo ocorre para o itálico:

Não Semântico: A palavra aviões está em itálico.
Semântico: A palavra avião está em ênfase.

Texto Marcado

A palavra helicóptero está marcada com a tag <mark>.

Texto Grande e Texto Pequeno

Estamos escrevendo um texto grande e um texto pequeno nesse paragrafo.

Texto Revogado

Podemos colocar um texto que foi revogado, ou seja, um texto que deve ser lido mas não deve ser considerado, como acontece nos códigos da lei.

Texto Inserido

Podemos marcar um texto como inserido, para indicar que este texto foi inserido, por exemplo, como uma retratação ao texto revogado.
