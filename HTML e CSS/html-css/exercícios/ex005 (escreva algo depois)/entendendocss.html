INTRODUÇÃO AO CSS

Um código CSS tem a seguinte forma: seletor {propriedade1= valor1; propriedade2= valor2; ...}

Regra CSS: é um conjuntos de propriedades associadas a um seletor. Por exemplo, se decidirmos que os paregrafos devem ter fonte amarela e background preto, então devemos escrever o seguinte código para implementar

p {color: yellow; background-color: black;}
fazendo isso, conseguimos o efeito visto nesta página.

Perceba que p é o seletor que representa o paragrafo do html, pois o paragrafo é <p>
outra coisa a se notar é que a sitaxe do CSS é extremamente diferente da sintaxe do HTML, exemplo: diferentemente do que ocorre em HTML, o valor da propriedade não é representado com aspas " " em CSS. Se houver algum erro ou qualquer outro problema no código, certifique-se de que o problema não é caso de uma confunção de sintaxe.
