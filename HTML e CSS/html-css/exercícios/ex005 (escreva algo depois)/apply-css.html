Como o CSS é aplicado a uma página?

Existem três formas de aplicar o CSS em uma página:

(Inline) Em que nos aplicamos a folha de estilo diretamente a um elemento do HTML. Para aplicar o CSS dessa forma, devemos, na abertura do elemento HTML, escrever a chave style e então dar o seu valor conforme mostra o exemplo: <h1 style="color= blue; background-color= black;"> </h1>. Esta é a pior forma de se aplicar o CSS. Evite, sempre que possível, usá-lo desta maneira. Os problemas dessa forma de implementação é que além de ser o exato oposto das boas práticas, ele é extremamente improdutivo, pois se houver a necessídade de se alterar o estilo de um elemente, teremos que mudar todos os elementos do mesmo tipo. A manutenção se torna quase impossível. Outro problema é que estamos misturando os códigos HTML e CSS o que torna o código difícil de ler e mais fácil de se confundir. Infelizmente, em alguns momentos, só temos permissão de alterar o elemento <body> do HTML, portanto, precisamos recorrer a esta forma;

(Internal Style Sheet) Em que o código CSS é escrito entre as tags <style> e </style> no próprio documento html. A vantagem dessa forma de implementação é que você pode modificar uma página específica diretamente e também que, se você não tiver acesso ao arquivo CSS que dá o estilo à página, então você pode sobre escrever o estilo com esse tipo de implementação. O problema é que a manutenção do sistema fica muito mais difícil e se houver mudanças em uma página, então será necessário alterar todas as outras manualmente diminuindo gravemente a produtividade;

(External Style Sheet) Em que o código CSS é escrito em um arquivo próprio para ele, isto é, em um arquivo de extesão ".css". É, de longe, a forma mais produtiva de se escrever CSS, pois o estilo criado pode ser usado em qualquer arquivo HTML. Para implementar o CSS de uma folha externa é necessário usar a tag <link>
